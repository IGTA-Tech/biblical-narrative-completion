## üèÜ Quality Standards (A+ Grade = 97+/100)

### Required Components for Each Contextual Analysis

Every section MUST include these 6 components:

#### 1. **Narrative Summary** (150-250 words)
- What happens in these chapters
- Key events, characters, plot points
- Clear, engaging storytelling

#### 2. **Historical & Archaeological Context** (250-400 words)
**Critical Requirements:**
- ‚úÖ Name SPECIFIC artifacts (steles, inscriptions, tablets, ostraca)
- ‚úÖ Name SPECIFIC archaeological sites (tells, excavations)
- ‚úÖ Cite SPECIFIC ancient sources (Mesopotamian texts, Egyptian records)
- ‚úÖ Reference SPECIFIC scholars and excavations (e.g., "Kathleen Kenyon's excavations at...")
- ‚úÖ Present MULTIPLE scholarly perspectives on dating
- ‚úÖ Balance traditional and critical scholarship
- ‚úÖ Distinguish fact from interpretation

**Examples of A+ Quality Citations:**
- "The Merneptah Stele (c. 1208 BCE) provides the earliest extra-biblical reference to Israel..."
- "Excavations at Tell es-Sultan (ancient Jericho) by Kathleen Kenyon (1952-1958)..."
- "The Dead Sea Scrolls (11QPsa) preserve Psalm 145 with variant readings..."

#### 3. **Cultural Practices Reflected** (200-350 words)
- Ancient Near Eastern customs
- Daily life, social structures
- Religious practices
- Family dynamics
- Economic systems

#### 4. **Political Situation** (200-350 words)
- Geopolitical context
- Kings, empires, vassal relationships
- Treaties and alliances
- Historical events
- Power dynamics

#### 5. **Theological Themes** (250-400 words)
- Major theological concepts
- Divine character and actions
- Covenant theology
- Redemptive history
- Spiritual principles

#### 6. **Connection to Broader Narrative** (250-400 words)
- How it fits the overall biblical story
- Connections to other books
- Typological patterns
- New Testament fulfillment (where applicable)
- Canonical significance
